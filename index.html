Scribbl.io Custom Preparer

This is a python project I made for those that want to let their friends make lists of custom words for the popular game skribbl.io.

  • You can find my original python script here
  • You can find my windows executable here, intended for Windows 10 64-bit computers (6/3/2020)
I figured since my friends don't necessarily sit down together to generate our custom words, it was pretty useful to instead make a quick and easy way to combine our own lists into a single, combined, and succinct copy-and-paste-able paragraph for the game.

A Discord Bot

This is a (at the time) working discord bot made from python. It's primarily a music bot, but it has a lot of other possible features, some built into it by the original maker and some added by me. Since it's very old and I was very new to it (This was both my introduction to python and my first step away from IDEs), this is probably an obsolete relic, but here it is nevertheless.
The original maker was named ASexualRhino.
